# DXMarsXlog

## 重要提示
- 本库仅支持真机设备（arm64架构）
- 不支持模拟器运行
- 最低支持 iOS 12.0

## 安装
DXMarsXlog is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'DXMarsXlog'
```
